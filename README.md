# Low Latency Project

A C++ project focused on low-latency performance optimizations.

## Features

- High-performance C++20 implementation
- Low-latency optimizations with compiler-specific flags
- Cross-platform support (GCC, Clang, MSVC)
- CMake build system

## Building

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Requirements

- CMake 3.15 or higher
- C++20 compatible compiler (GCC, Clang, or MSVC)
